# 文案策略备忘 · 《沉默的计数》

## 标题命名依据（调研摘要）

### 普利策解释性/数据报道
- **数字+阈值**：《2°C: Beyond the Limit》
- **动词/后果**：《Burned》《Terror on Repeat》
- **把抽象变具体**：算法、保险、种群曲线——标题指向「被掩盖的系统」

### 南方周末特稿
- **冒号结构，时间+命运**：《作别白鲟：苦追十九载，难逃悲剧》
- **设问直击普通人**：《白鲟灭绝！一个物种的灭绝对普通人意味着什么》
- **引语+反差**：《野生种「一龟难寻」》
- **四化原则**：主题事件化 → 事件故事化 → 故事人物化 → 人物性格化

### 人物 / 南方人物周刊
- **时间+主体**：《九十李泽厚 最后的访谈》
- **命运感**：「记录我们的命运」「重新打量每个生命」

### 澎湃美数课
- **行动+比喻**：《想要衡量生物多样性，我们来给全球动物称一下体重》
- **把宏观数据翻译成可感知动作**

## 本作标题体系

| 层级 | 文案 | 逻辑 |
|------|------|------|
| 主标题 | 沉默的计数 | 保留原品牌；「计数」双关监测数据与生命消逝 |
| 英文 | Silent Count | 简洁、新闻标题感 |
| 导语 | 当指数跌至 13，有些名字只剩下 2 | 数据集两个极端锚点（LPI 2020 / 最濒危物种） |
| 第一幕 | 半世纪坠落 | 时间跨度 + 动词 |
| 第二幕 | 大洋洲去了哪里 | 地理具象（-88.9% 最痛区域） |
| 第三幕 | 名字后面，只剩个位数 | 南方周末式命运句 |
| 第四幕 | 从「无危」到极危 | IUCN 光谱，保留专业感 |
| 第五幕 | 南极的身体在报警 | 微观身体信号，与宏观曲线对照 |
