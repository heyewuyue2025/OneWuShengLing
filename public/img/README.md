# 配图放置说明

将 AI 生成的图片按以下文件名放入此目录：

| 文件名 | 用途 | 建议尺寸 |
|--------|------|----------|
| `hero-bg.jpg` | 首页全屏背景 | 1920×1080 |
| `act1-global.jpg` | 第一幕装饰（可选） | 800×600 |
| `act2-regions.jpg` | 第二幕装饰（可选） | 800×600 |
| `act3-species.jpg` | 第三幕物种氛围图 | 800×600 |
| `act4-spectrum.jpg` | 第四幕装饰（可选） | 800×600 |
| `act5-penguin.jpg` | 第五幕企鹅特写 | 800×600 |

图片缺失时页面仍可正常运行（使用渐变背景）。
